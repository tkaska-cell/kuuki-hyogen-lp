# 空気を味方につけて話せる表現講座 — LP

前野太志（まえのひろし／笑顔ナビゲーター）の表現講座ランディングページ。

- 公開URL: https://tkaska-cell.github.io/kuuki-hyogen-lp/
- 構成: `index.html` 1ファイル完結（CSS・JS内蔵、外部依存はGoogle Fontsのみ）

## マーケティングの型

エンパシーライティング（読者の感情を先回りして言語化）＋ 新PASONA を土台に構成：

1. **共感** — 「人前で消耗していませんか」読者の内的独白を言語化
2. **問題の再定義** — 「あなたのせいではない／空気との距離感の問題」
3. **やらない／やること** — 具体的な“ひと言”で差別化
4. **USP** — ★最大の尖り：「話し方の先生ではなく、今も毎週現場に立つ現役MC」
5. **向き不向き → 受講生の声 → オファー → 個別相談 → プロフィール → FAQ**

## 公開前に差し替える3か所

| 箇所 | 現状 | 対応 |
|---|---|---|
| **① 個別相談ボタンのリンク** | `#`（仮） | `index.html` 内 `id="line-cta"` と `.nav-cta` の `href` を、公式LINE／予約フォームのURLに差し替え |
| **② ひろしさんの写真** | プレースホルダー | `.about-photo` の中身を `<img src="...">` に差し替え（4:3推奨） |
| **③ 受講生の声（3名）** | 構成イメージのサンプル | 実際の受講生の声に差し替え。HTMLコメントで該当箇所を明記。**景品表示法・ステマ規制対策のため、実在しない声を実績として公開しないこと** |

## 更新方法

`index.html` を編集 → `git commit` → `git push` で数十秒後に反映される。
