# 空気を味方につけて話せる表現講座 — LP

前野太志（まえのひろし／笑顔ナビゲーター）の表現講座ランディングページ。

- 公開URL: https://tkaska-cell.github.io/kuuki-hyogen-lp/
- 構成: `index.html` 1ファイル完結（CSS・JS内蔵、外部依存はGoogle Fontsのみ）＋ `assets/` 画像

## マーケティングの型

エンパシーライティング（読者の感情を先回りして言語化）＋ 新PASONA を土台に構成：

1. **共感** — 「人前で消耗していませんか」読者の内的独白を言語化
2. **問題の再定義** — 「あなたのせいではない／空気との距離感の問題」
3. **やらない／やること** — 具体的な“ひと言”＋SVG図解で差別化
4. **USP** — ★最大の尖り：「話し方の先生ではなく、今も毎週現場に立つ現役MC」
5. **講座の雰囲気 → 向き不向き → 受講生の声 → オファー（モニター価格）→ 個別相談 → プロフィール → FAQ**

## 価格（アンカリング）

通常 98,000円 → **期間限定モニター価格 50,000円（税込）**。
「受講後の感想協力とひきかえに割引／枠が埋まれば通常価格に戻る」という募集ロジックを併記。

## 公開前に差し替える残り2か所

| 箇所 | 現状 | 対応 |
|---|---|---|
| **① 個別相談ボタンのリンク** | `#`（仮） | `index.html` の `id="line-cta"` と `.nav-cta` の `href` を、公式LINE／予約フォームのURLに差し替え |
| **② 受講生の声（3名）** | 構成イメージのサンプル | 実際の受講生の声に差し替え。**景品表示法・ステマ規制対策のため、実在しない声を実績として公開しないこと**（サイト上にも注記済み） |
